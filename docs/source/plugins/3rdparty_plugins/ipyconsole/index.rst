IPyConsole
==========

The *IPyConsole* plugin adds an `IPython <https://ipython.org/>`_ shell to QGIS.

*IPython* is an extremely powerfull Python shell with a lot of cool features
that will boost user's productivity, like:

* full introspection capabilities
* *TAB*-completion on variables, methods, functions, modules and even filesystem paths
* inline docstring help for all methods and functions
* full browseable history of commands
* can run OS commands from the shell
